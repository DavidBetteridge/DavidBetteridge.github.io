I currently work as a Senior Software Developer for a startup called éXō Fleets. We use data science and machine learning to make fleets of commercial vehicles more efficient. Day to day I use a range of technologies which include C#, Postgres, Azure, Python and React.js.

I am a co-founder of the York Developers charity, which arranges technical meetups and events in York. These include a monthly Code Dojo where attendees pair up and work on a new problem each month. Additionally, I have spoken a number of times at community usergroups, and have both ran workshops and volunteered for conferences such as NDC London and NDC Security Oslo.

In my free time I'm a keen runner and have competed in several ultra-marathons. These include 'The Line', the Winter Spine Race, the Hardmoors 200, and the Arc of Attrition. More information about my running can be found here.
